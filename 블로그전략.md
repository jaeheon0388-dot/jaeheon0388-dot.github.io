## 핵심 키워드 매트릭스
글 제목은 아래 조합으로 만든다. 한 글에 한 조합만 (키워드 분산 금지).

**[지역] × [업종/품목] × [서비스]**

| 지역 | 업종/품목 | 서비스 |
|---|---|---|
| 서울 | 미용실 수건 | 세탁 업체 |
| 경기남부 | 헬스장 수건 | 세탁 비용 |
| 수원 · 성남 · 용인 | 사우나 수건/찜질복 | 렌탈 |
| 안양 · 안산 · 화성 | 마사지샵 타월 | 위탁 세탁 |
| 평택 · 오산 · 군포 | 헬스복 | 수거 배송 |
| 강남 · 강서 등 서울 구 단위 | 업소용 수건 | 대량 세탁 |

예: "서울 미용실 수건 세탁 업체 고르는 법", "수원 헬스장 수건 세탁 비용, 자체세탁과 비교하면"
※ 지역명은 실제 배송 가능한 도시부터 우선 발행. 도시별 글은 내용이 80% 같아도 검색에서는 별개로 잡힌다 — 단, 문장을 그대로 복붙하지 말고 도시별로 다시 쓸 것 (중복 문서 판정 방지)

## 글 유형 4가지 (발행 비율)
1. **비용/비교형 (40%)** — "OO 세탁 비용", "자체세탁 vs 위탁" → 검색량 최다, 문의 직결
2. **선택 가이드형 (30%)** — "OO 세탁 업체 고를 때 확인할 5가지" → 신뢰 구축
3. **문제 해결형 (20%)** — "수건 쉰내 없애는 법", "수건 수명 늘리기" → 롱테일 유입
4. **지역/사례형 (10%)** — "[지역] 납품 사례", "수거 배송 권역 안내" → 지역 검색 장악

## SEO 글 규칙 (모든 글 공통)
- 분량 1,500자 이상, 소제목(h2) 3~5개
- 제목·첫 문단·h2 하나에 핵심 키워드 자연스럽게 포함
- 글 끝에 FAQ 2~3개 (검색 롱테일 흡수)
- 모든 글에 문의 CTA 박스 (전화 + 카카오채널)
- meta description 80자 내외, 키워드 포함
- 새 글 발행 시 sitemap.xml에 URL 추가

## 발행 파이프라인 (자동화 구조)
1. Claude가 키워드 매트릭스에서 다음 조합 선택 (발행기록.md 확인, 중복 방지)
2. `posts/YYYY-MM-DD-슬러그.html` 생성 (templates/post-template.html 기반)
3. index.html 최신 글 목록 갱신 + sitemap.xml에 추가
4. `발행기록.md`에 기록
5. git commit & push → GitHub Pages 자동 반영
→ 이 5단계 전체를 매일 스케줄 작업으로 실행 가능
